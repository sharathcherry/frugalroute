# FrugalRoute — Local-Model Benchmark

> Which small open-weight model routes best? Every model runs through the **full
> FrugalRoute pipeline** (tools → local model → verify → gate → remote) on a single
> **AMD Instinct MI300X** (vLLM/ROCm) and is scored on the labeled eval set at its
> tuned accuracy threshold. **Ranked by remote tokens at accuracy ≥ floor — lower is better.**
>
> Thesis: **a small model + tools + verification matches big-model accuracy at a
> fraction of the tokens, latency, and VRAM.** The 32B is included only as a *control*.

## What we measure (per model)

| Metric | Meaning |
|---|---|
| **Local hit-rate** | % of tasks answered at **0 remote tokens** (tool tier + accepted local) |
| **Remote tokens** | total paid tokens over the eval set — *the leaderboard metric* |
| **Accuracy** | overall + per-category (classification / extraction / math / qa / summarization / reasoning); must be **≥ accuracy floor** to count |
| **Latency** | avg ms / task on the MI300X |
| **VRAM** | peak GPU memory (`rocm-smi`) |

**Ranking key:** lowest **remote tokens** at **accuracy ≥ floor**; tie-break by latency, then VRAM.

## Reproduce

Requires the FrugalRoute repo + Ollama (or vLLM) serving on the target GPU, and a remote
(`REMOTE_PROVIDER=fireworks` + key) in `.env`.

```bash
# runs every model in the slate through the full pipeline
bash eval/run_benchmarks.sh            # pull → harness (tune threshold) → run → record
python eval/make_readme_table.py       # writes eval/benchmark.json + this table + eval/pareto.png
```

Per-model loop (what the script does):
```bash
ollama pull "$TAG"
LOCAL_MODEL="$TAG" python eval/harness.py                                  # fit calib + threshold
LOCAL_MODEL="$TAG" python run.py --input eval/tasks.jsonl --output out/$TAG.jsonl
# record local% / remote_tokens / per-category accuracy / latency / VRAM -> eval/benchmark.json
```

**Environment:** AMD Instinct MI300X (192 GB, ROCm) · local via vLLM/Ollama · remote = Fireworks
`gpt-oss-120b`. Eval set: `eval/tasks.jsonl` (labeled, 6 categories). Accuracy floor: 0.80.
